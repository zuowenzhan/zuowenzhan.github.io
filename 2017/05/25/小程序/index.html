小程序讲解（一）

发表于 2017-05-25 | | 阅读次数
微信小程序出来有一段时间了，并没有怎么关注，已不是太火。然而忽然间公司让我开发微信小程序，为了完成任务，也为了学习更多的知识。于是开启了我的小程序探索之路，经过简单的学习之后，其实感觉前端人学起来会更容易些，而不是我们移动端来研究。小程序和前端像似，但对于一个程序员而言，只要掌握了一门语言，在学习其他的都不是问题。

1.目录讲解（开发工具到微信公众平台下载）

1.1 app.js、app.json、app.wxss 这三个文件。其中，.js后缀的是脚本文件，.json后缀的文件是配置文件，.wxss后缀的是样式表文件。

1.2 页面中 .js .wxml文件是必不可少的，其中.js文件是页面逻辑，.wxml是布局文件。而.json .wxss都不是必须的，页面中的会覆盖项目的文件。

2.控件讲解

2.1 Text控件显示

2.1.1 默认
<text>我是内容</text>
2.1.1 动态显示 textId 是唯一标识id
<text>{{textId}}</text>

在.js中

data: {
 text : "你好棒",
},
2.2 Button点击动态改变text内容
<button bindtap="onclick"> 这是Button </button>
<text>{{text}}</text>

在.js中代码

onclick:function(){
  this.setData({ text: "棒棒的。。。。")
 }
2.3 点击Button显示隐藏text，默认是显示
<view wx:if="{{show}}">{{text}}</view>

在.js中

data: {
text : "你好棒",
show:false,
},

 onclick:function(){
  var isShow=this.data.show,
  this.setData({ text: "棒棒的。。。。",show:!isShow})
 }
2.3.2交替显示文本内容，用if….else…
<view wx:if="{{show}}">{{text}} 1 </view>
<view wx:else >{{text}} 2 </view>

在.js中

data: {
text : "你好棒",
show:false,
},

 onclick:function(){
  var isShow=this.data.show,
  this.setData({ text: "棒棒的。。。。",show:!isShow})
 }
2.4 for循环
<view wx:for="{{news}}" wx:for-item="itemx" wx:for-index="ix">
{{ix}} - {{itemx}}
</view>
news—–对for循环命名
itemx —-对item重命名，默认是item
ix—-小角标

在.js中动态复制

data:{
news : ['aaa','bbb','ccc','ddd']  
}

动态删除

var newsdata = this.data.news;
newsdata.shift()
this.setData({text:"这是一个新的内容...",news:newsdata})
2.5 引入布局模板

新建一个文件夹，名字随便取，写布局文件如下：
新建头部布局命名为header.wxml 内容如下

<text> 
 这里是头部的组件...
</text>

引入：

<include src="../templates/header" />

第二种引入，内容如下：

 <template name="footer1">
       这是底部内容1
 </template>
<template name="footer2">
   这是底部内容2 
 </template>

引入：

<import src="../templates/footer" />

动态改变内容

<template name="footer1">
    这是底部内容1  - {{text}}
 </template>
<template name="footer2">
    这是底部内容2  - {{text}}
</template>

引入

<template is="footer2" data="{{text:'导入设置的内容...'}}" />
