AndroidStudio中git版本控制

发表于 2017-05-11 | | 阅读次数

在之前的项目中使用svn版本控制工具，现在公司统一使用git,在刚接触git时，同事简单介绍了一下git命令行提交代码，顿时感觉云里雾里，感觉太麻烦了。于是开始了我的使用搜索之路，发现AS本身就可以git操作，顿时明朗起来。

1.在工程的.gitignore中添加忽略，不得不说和svn添加忽略一样一样的

删除原来默认的添加一下代码

 .gradle
 .idea
 build
 *.iml
/local.properties

2.添加git客户端安装路径 如图：

3.点击VCS—->Enable Version…..—>选择Git—>OK

4.点击VCS—->Import into…..–>Share prioject….—>取个名字，点击share,点击oK

5.下面把鼠标放到左侧project任意文件上，然后Ctrl+A，选中所有文件，右键，git -> add （全部变为绿色）

6.点击导航栏VCS向上箭头commint提交，提交成功之后颜色变为正常

7.在左边导航栏任意位置右键GIt–>Repository—>push

8.输入提交URL,输入用户名和密码

9.push 如图：

10.以后push和pull直接点导航栏上VCS就可以，这是成功创建了一个master版本
