教你怎么搭建个人博客

发表于 2017-04-19 | | 阅读次数

一、用到软件，前提安装软件

1.安装Node.js软件，都是下一步

2.安装git软件下载地址

3.安装hexo

3.1右键运行git，打开窗口

命令：$ npm install -g hexo-cli

3.2、检验是否安装成功了

命令：hexo -v

4、NexT模板简介

NexT 拥有丰富而简单的配置,结合第三方服务,打造属于您自己的博客 主题配置 NexT 坚持将复杂的细节隐藏,提供尽量少并且简便的设置,保持最大限度的易用性。

>
模板生成地址：
访问地址

5、Hexo简介

说明：NexT是基于Hexo的所以，还要去了解Hexo
Hexo介绍：
Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

访问地址

使用：如果使用Hexo，还需要使用Node.js以及git

二、步骤:

参考文档

参考博客

1、在计算机任意盘创建一个MyBlog文件夹

命令：

1.1 cd d:MyBlog

用途：找到这个文件夹

1.2、hexo init

用户：在当前文件夹配置hexo
运行完命令稍等一下：
查看MyBlog文件夹会生成一些文件：

3、hexo new “Hello”
通过命令创建一个Hello.md文件
运行完之后会提示：INFO Created: E:\blog\source_posts\Hello.md
意思是：在当前文件夹下，创建了一个makedwon的文件

4、打开文件:
我们只需要在—的下面编写自己的博客内容就可以了。

5、生成预览文件：
命令：hexo generate

6、预览：
命令：hexo server
作用：开启本地的服务器
预览地址：
http://localhost:4000/

6、配置信息;

预览后发现：博客名，作者等信息都是hexo默认的。
找到_config.yml文件，在里面修改掉即可，当然，还可以根据hexo的配置文档去配置。

六、配置主题使用NEXT主题：

1、克隆next样式到站点文件夹
命令：
1、cd d:Mybolg
2、 git clone https://github.com/iissnan/hexo-theme-next themes/next
它会自动下载：

验证：通过git下载之后，会有next文件夹出现

2、启动主题：
1、找到Myblog最外层的文件夹，找到_config.yml文件
2、找到字段theme

3、修改landscape为next

4、验证主题：
到此，NexT 主题安装完成。下一步我们将验证主题是否正确启用。在切换主题之后、验证之前， 我们最好使用 hexo clean 来清除 Hexo 的缓存。

清除hexo缓存:
命令：hexo clean

5、验证结果：

6、这个时候已经能看见默认的next的主题了

7、切换next主题：

1、外观样式：
找到next文件夹下的 .config.yml

2、搜索scheme
可选参数:
Muse - 默认 Scheme，这是 NexT 最初的版本，黑白主调，大量留白
Mist - Muse 的紧凑版本，整洁有序的单栏外观
Pisces - 双栏 Scheme，小家碧玉似的清新

8、设置语言：

设置语言：在最外层的.config.yml中设置
language: zh-Hans

9、设置菜单：
菜单是在主题下面的.config.yml中设置

字段：menu
menu:
home: /
archives: /archives

#about: /about

#categories: /categories
tags: /tags

#commonweal: /404.html

上面的几项内容，需要直接放开就可以了。

10、作者昵称
配置地点：最外层配置文件
字段：author

11、设置链接;
配置位置：主题配置文件
字段：social

12、设置头像

七、上传到github

配置git账号：
git config –global user.name “yourname”
git config –global user.email “youremail”
github注册的邮箱

部署：找到最外层的.config.yml，配置如下参数
http://www.tuicool.com/articles/aQriym3
1 deploy:
2 type: git
3 repository: https://github.com/cnfanhua/cnfanhua.github.io.git

报错：
Cannot find module ‘hexo-util’
解决:npm install hexo-util

生成ssh：

ssh-keygen -t rsa -C “xxxxxx.com”
github注册邮箱

生成文件id-ras,找到文件路径打开复制全部内容，到github的setting中添加key

极光推送快速集成
